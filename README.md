# StudentMangement

A Spring Boot project for managing student records with CRUD operations and H2 in-memory database.

## 🚀 Features
- Create, read, update, and delete student records
- REST API endpoints using Spring Boot
- Integration with H2 in-memory database
- H2 console for database inspection
- Example endpoints tested with Postman

## 🛠️ Tech Stack
- **Java 17**
- **Spring Boot 4.0.6**
- **Spring Data JPA**
- **H2 Database**
- **Maven**

## 📦 Requirements
- JDK 17+
- Maven 3.8+
- IDE (STS, IntelliJ, Eclipse)
